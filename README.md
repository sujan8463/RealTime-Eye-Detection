Features:
      Real-time face detection using webcam      
      Detects eyes within the detected face region      
      Displays rectangles around faces and eyes      
      Lightweight and beginner-friendly

Tech Stack:
    Python 3    
    OpenCV    
    Haar Cascade Classifiers
